TP2 – Procesamiento Digital de Imágenes

Este repositorio contiene los scripts desarrollados para resolver el Trabajo Práctico Nº2 de la materia Procesamiento Digital de Imágenes. Incluye el código del Problema 1 (detección y clasificación de monedas y dados) y el Problema 2 (detección de patentes y segmentación de caracteres), junto con las imágenes necesarias para ejecutar ambos programas.

Para correr los scripts, solo es necesario abrir la carpeta del TP2 en cualquier entorno con Python instalado. Luego, ejecutar cada archivo .py desde la terminal. Ambos scripts cargan automáticamente las imágenes incluidas en las carpetas correspondientes y muestran los resultados por pantalla.
